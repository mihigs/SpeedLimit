**Speed Limit**

## Usage

1. Clone this repository:

   ```shell
   git clone https://github.com/yourusername/VariableSpeedLimitCalculator.git

2. Build and run the project in your preferred C# development environment (e.g., Visual Studio).

3. You can customize the journey length and speed limits in the Program.cs file to test different scenarios.

4. Run the unit tests in the Tests project to ensure the correctness of the VariableSpeedLimit class. You can use NUnit or your preferred testing framework.
